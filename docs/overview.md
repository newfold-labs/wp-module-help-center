# Overview

**wp-module-help-center** provides the help center experience for Newfold brand plugins, including search via Typesense. It registers with the Newfold Module Loader and depends on wp-module-ai, wp-module-data, wp-module-features, and typesense/typesense-php. Maintained by Newfold Labs. Distributed via Newfold Satis.
