# Getting started

Prerequisites: PHP 7.3+, Composer. The module requires wp-module-ai, wp-module-data, wp-module-features, typesense/typesense-php.

```bash
composer install
composer run test
composer run lint
composer run fix
```

See [integration.md](integration.md) for using in a host plugin.
